# USE
1) Start the server: “./main [-u]”, default - TCP connection, [-u] – for UDP connection.
2) Connect the client: “./main [IP-address] [port] [-u]”, default - TCP connection, [-u] – for UDP connection.
3) Follow the printed rules.
